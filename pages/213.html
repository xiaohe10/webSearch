2014年的脚步声临近了，回首2013年的国内乘用场，南北表现优异，交出了一份不错的销量答卷。而在接下来的一年里，本田又将有什么新产品？今天我们梳理了下此前获得的消息，下面给各位读者一一介绍。 广汽本田 2013年广州车展期间，广汽本田执行副总经理郁俊在接受网易汽车采访时表示明年将会推出4款新产品来丰富产品线，更好地满足中国消费者的需要。而结合最新的消息看，这4款车型基本有了眉目。车型：全新小型SUV预计上市时间：2014年下半年新车信息：1.广本&东本都将推出 2.时尚设计 3.将搭1.5L/1.8L动力 今年广州车展上，本田又带来了一款即将面向中国市场推出的全新SUV车型，其目前还处于概念阶段，名为“Concept V”。而据本田中国本部长仓石诚司透露，本田将以这款概念车为基础，分别开发两款新产品，并于明年以后分别投放广汽本田和东风本田。换言之，Concept V的量产版本绝不只有1个版本，甚至国产车型是否照搬Vezel的设计也依然存疑。 本田Concept V概念车 ● 外观：缩小版的CR-V 本 田Concept V概念车定位于未来主力的城市通勤小型化SUV，车身长度控制在4300mm以内，轴距则达到2550mm，由于采用了高顶设计，因此内部空间表现会很出 色。从外部设计来看，Concept V概念车引入了本田新SUV车型的外观设计，分段式的进气格栅显得非常大气和动感，而锋利的前车灯以及LED技术的引入增加了不少科技感。 /44 | | | Concept V从图片不难看出，Concept V尾部造型基本照搬了Urban SUV的设计，甚至轮毂造型也一模一样。事实上，早前这款本田小SUV量产版已经在11月20日举行的东京车展正式发布。这款概念车在国内首发，更多起到的是预热的意义。 ● 量产版已发布 国产版或略有差异 本田Vezel量产车已经在东京车展发布，与Concept V相比，量产版进气格栅变得更加传统，标准的大灯布局也同概念车的多LED光源不同。具体来看，车型分段式的进气格栅显得非常大气和动感，而锋利的前车灯以及LED技术的引入增加了不少科技感。此外，前后保险杠、后视镜等都有更为正常的设计，独特的隐藏式后车门把手设计得到了保留，这也让车型显更为紧凑和独特。 车身尺寸官方依旧没有给出任何数据，根据此前的消息表明，车型长度不足4300毫米，轴距2550毫米，由于采用了高顶设计，因此内部空间表现会很出色。 内饰部分来看，依旧保持了本田一贯的科技含量极高的设计，新家族式的方向盘、新式的仪表盘以及朝向驾驶员侧的中控台设计都比较独特，而类似于飞机推杆的变速箱档杆显得非常漂亮。 Vezel 同本田飞度一样采用GE平台打造，除了增加轴距以外，还采用了大量铝合金材质以进一步降低车身重量，从而实现更高的燃油经济性。在动力方面，Vezel同 飞度也完全一致，搭载Earth Dreams系列的1.3升和1.5升两款汽油发动机，并提供同混合动力飞度一样的混合动力版本。 /24 | | | 小SUV新弄潮儿
